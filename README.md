# Around The U.S. — Full Stack Project

## Project Description

This is a full stack web application developed as part of the TripleTen Web Development program.
Users can register, log in, edit their profile, update their avatar, create cards, like cards, and delete their own cards.

The project includes:

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Deployment:** Google Cloud VM + Nginx + PM2

---

## Live Project

- **Frontend:** http://charlesaround.mooo.com
- **Backend API:** http://api.charlesaround.mooo.com:3000

## Running the Project Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
